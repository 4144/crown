virtual bool crown::File::is_valid ( )
pure virtual

Returns whether the file is valid.

A file is valid when the buffer where it operates exists. (i.e. a file descriptor is attached to the file, a memory area is attached to the file etc.)

Implemented in crown::NullFile, crown::DiskFile, and crown::ApkFile.
